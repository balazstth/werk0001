
01. REQUIREMENTS
02. START

01. Runtime requirements / developed using:

- JRE 14
- Groovy 3

It is possible, that earlier Java and Groovy runtimes also suffice.

02. Start the application with:

- start.bat (or equivalent script on your platform), that uses the packaged Windows JRE and Groovy binary.
- start-without-bundled-runtime.bat (or equivalent script on your platform), that uses your system JRE and Groovy binary.
  JAVA_HOME and GROOVY_HOME should be set properly, also the system PATH to their bin subfolders.
